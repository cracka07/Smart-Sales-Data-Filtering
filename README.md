# Smart-Sales-Data-Filtering
Excel VBA system that dynamically filters sales data by date, store, salesperson and product, enabling fast analysis of sales datasets directly in Excel.

<img width="888" height="434" alt="data filtering" src="https://github.com/user-attachments/assets/9c4fd91a-a113-4829-8604-e83dbd934a17" />

## Features

- Dynamic filtering of sales data
- Filtering by:
  - Date
  - Store
  - Salesperson
  - Product
- Automatic update of filtered results
- Optimized VBA logic for fast data processing

## Technologies Used

- Microsoft Excel
- VBA (Visual Basic for Applications)
